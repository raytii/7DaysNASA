# NASA 7-Day NEO Data Archive

An immersive 3D generative archive that translates seven days of NASA Near-Earth Object (NEO) data into a flowing topological network of light and motion.

## Project Concept
This sketch functions as a "Celestial Archive," utilizing raw astronomical data to construct a visceral bridge between complex mathematics and human sensory intuition. By mediating data through algorithmic topology, it transforms cold metrics like miss distance and velocity into an aesthetic "Cosmic Web."

## Key Features
- **Live Data Binding:** Connects to NASA's NeoWs API to fetch real-world asteroid activity.
- **Topological Fluidity:** Uses 3D Perlin Noise and Bezier interpolation to create smooth, organic "strands."
- **Depth-of-Field Simulation:** Implements Z-depth mapping to control color (Cyan to Magenta) and transparency, creating a strong longitudinal perspective.
- **Performance Optimized:** Pre-calculates 3D paths to ensure 60fps rendering in a browser environment.

## Tech Stack
- [p5.js](https://p5js.org/) - Creative Coding Library
- [NASA Open APIs](https://api.nasa.gov/) - Data Source
- WebGL - 3D Rendering Engine

## How to Run
1. Clone the repository.
2. Open `index.html` in any modern web browser (Chrome recommended).
3. Use the mouse to rotate and scroll to zoom into the data network.
